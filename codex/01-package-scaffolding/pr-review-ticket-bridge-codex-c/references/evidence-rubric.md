# Evidence Rubric

Use this rubric while reviewing PR Review Ticket Bridge.

- Confirm the scope and claimed success condition first.
- Collect direct evidence for input normalization, deterministic structure, and validation gates.
- Separate observed facts from interpretation.
- Rank issues by user impact and rework cost.
