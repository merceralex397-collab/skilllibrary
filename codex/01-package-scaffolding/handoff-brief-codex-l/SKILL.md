# Handoff Brief

Use this skill whenever the repo needs a restartable handoff.

## Required outputs

- a top-level `START-HERE.md`
- the current ticket or next recommended ticket
- links to canonical docs
- validation status
- known risks and open questions

Use `references/handoff-checklist.md` and the starter template in `assets/templates/START-HERE.template.md`.

This skill is a closeout and restart refiner, not a scaffold entrypoint.
