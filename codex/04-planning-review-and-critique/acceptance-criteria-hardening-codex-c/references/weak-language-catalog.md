# Weak Language Catalog

Flag phrases like these unless they are made observable:

- works well
- user-friendly
- intuitive
- robust
- scalable
- performant
- complete
- seamless
- properly handles
- as expected

## Replacement strategy

Replace the adjective with one of:

- a visible behavior
- a measurable threshold
- a specific failure response
- a named proof source

Example:

- weak: "The search feels fast."
- stronger: "Search results render within the agreed response target and the empty-state message appears when no results match."
