# Conversion Patterns

Convert assumptions into actionable artifacts.

## Unknown dependency behavior

Convert to:

- a validation task
- a fallback path
- a monitoring requirement

## Unknown approval or ownership

Convert to:

- an approval gate
- an owner assignment
- an escalation path

## Unknown data quality

Convert to:

- a discovery query
- a sample audit
- a migration dry-run

## Unknown user behavior

Convert to:

- usability validation
- staged rollout
- instrumentation for the risky path
